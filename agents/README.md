# Agents Directory

## Purpose
This directory houses the intelligent, autonomous AI agents of HospitalityAI. Built using multi-agent coordination frameworks (such as LangGraph), it implements cognitive loops, planners, executors, and system supervisors that coordinate tasks on behalf of guests, operators, and managers.

## Ownership
- **Owner**: Agent Platform Team (Antigravity AI Coding Agent)
- **Primary Domain**: Agent Coordination, Cognitive Orchestration, Tool Invocation

## Directory Structure
- `planner/`: Logic for parsing complex workflows into sub-tasks.
- `executor/`: Executes atomic tools and API commands.
- `supervisor/`: Monitors state, routes tasks, and manages human-in-the-loop escalations.
