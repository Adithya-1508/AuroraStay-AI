## 1. Purpose
Define the canonical package directories structure of HospitalityAI.

## 2. Directory Mappings
- `/shared/`: Common infrastructure, config parsers, custom errors, and logger.
- `/docker/`: Environment Dockerfile builds.
- `/scripts/`: Setup, healthcheck, and db seed automation scripts.
- `/tests/`: Project test suite structure.
- `/.github/`: Action workflows and templates.
