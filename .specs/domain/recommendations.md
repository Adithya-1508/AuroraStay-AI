# Spec: Recommendations Domain Aggregate

- **Status**: Ready
- **Owner**: Decision Intelligence Context Owner (Antigravity AI Coding Agent)
- **Date**: 2026-07-04

## 1. Purpose
Define properties of proposals generated by AI/ML algorithms, ensuring they are validated and structured.

## 2. Responsibilities
- Store recommended actions (price adjustments, VIP room upgrades).
- Record confidence scores and validation metrics.

## 3. Public Interfaces
```python
class Recommendation:
    def __init__(self, recommendation_id: str, rec_type: str, details: dict, confidence: ConfidenceScore):
        self.recommendation_id = recommendation_id
        self.rec_type = rec_type  # e.g., 'RoomUpgrade', 'RateAdjustment'
        self.details = details
        self.confidence = confidence
        self.status = "Pending"
```

## 4. Invariants
- Confidence scores must remain between `0.0` and `1.0`.
