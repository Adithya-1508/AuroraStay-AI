# Dashboard Directory

## Purpose
Contains the frontend UI application. It provides visualization dashboards for guest analytics, reservation lists, operations calendars, dynamic pricing recommendations, and agent execution metrics.

## Ownership
- **Owner**: Frontend UI Team (Antigravity AI Coding Agent)
- **Primary Domain**: UI Components, State Management, Data Visualization

## Architecture Details
- Written in modern HTML5, CSS3, and JavaScript/TypeScript.
- Integrates with the API Platform using RESTful endpoints.
- Displays business metrics (occupancy, booking pace), operations metrics (housekeeping tasks, alerts), and ML metrics.
