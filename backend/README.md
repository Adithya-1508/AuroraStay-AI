# Backend Directory

## Purpose
This directory houses the application server bootstrapping code, environment configuration setups, dependency injection mappings, and global middleware structures.

## Ownership
- **Owner**: Backend Platform Team (Antigravity AI Coding Agent)
- **Primary Domain**: Server Bootstrap, App Lifecycle, Dependency Injection

## Key Responsibilities
1. Starting up the FastAPI server process.
2. Managing system startup and shutdown events (database pools, model warming).
3. Defining dependency injection configurations (mapping repository interfaces to actual database repositories).
