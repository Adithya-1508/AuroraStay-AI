# Knowledge Platform Abstraction Manual

Welcome to the HospitalityAI Retrieval-Augmented Generation (RAG) Knowledge Platform documentation manual.

This package provides a unified framework for document ingestion, text parsing, structural chunking, vector indexing with Qdrant, semantic search retrieval, NVIDIA reranking, citations provenance tracking, evaluation metrics, and re-indexing version control.
