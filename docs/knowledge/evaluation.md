# Retrieval Evaluation Manual

The `RetrievalEvaluator` metrics suite measures the quality and performance of semantic searches.

## Computed Parameters

- **Precision**: Percent of relevant retrieved chunks.
- **Recall**: Percent of found golden chunks.
- **Context Relevance**: Average vector similarity score.
- **Citation Accuracy**: Provenance tag validation rate.
- **Latency**: End-to-end execution duration in seconds.
