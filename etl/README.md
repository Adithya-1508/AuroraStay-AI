# ETL (Extract, Transform, Load) Directory

## Purpose
Contains data pipelines responsible for scraping, cleaning, extracting, transforming, and loading historical data (reservations, reviews, stays, operations logs) into relational and vector databases.

## Ownership
- **Owner**: Data Platform Team (Antigravity AI Coding Agent)
- **Primary Domain**: Data Pipelines, Ingestion, Normalization, Data Quality Checks
