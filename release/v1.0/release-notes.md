# HospitalityAI Release Notes (v1.0.0)

We are proud to release Version 1.0.0 (Gold Master) of the HospitalityAI (AuroraStay AI) Platform, an enterprise-grade AI-driven guest experience and operations suite.

## Release Features Summary

1. **Intelligent Booking Engine**: Automated availability calendar routing, promo discount matching, and Platinum/Gold priority room allocation upgrades.
2. **LangGraph Agent Workspace**: Structured chat assistant answering guest inquiries with citations mapping back to document collections.
3. **Structured AIOps Telemetry**: Structlog logging with injected correlation IDs, Prometheus metric counters, and Population Stability Index (PSI) drift indicators.
4. **Platform Security Shield**: JWT tokens, SHA-256 hashed API keys, sliding window rate limiters, ABAC context policies, PII anonymizers, and cryptographically chained audit trails.
5. **OCI Deployment Platform**: Multi-stage Dockerfiles, Kubernetes manifests, Helm packaging charts, and Terraform Infrastructure as Code.
